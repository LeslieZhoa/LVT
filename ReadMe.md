### How to run
1. environment<br>
   ```shell
   pip install -r requirements.txt
   ```
2. download model<br>
    ```shell
    cd utils
    bash download_weight.sh
    ```
3. run<br>
   follow LVT.py and find details<br>
   ```py
   python LVT.py
   ```
